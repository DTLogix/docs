# ПРОФІЛІ ГЕНЕРАЦІЇ ЛЕНДІНГІВ

## A. SaaS Landing Page Profile

### Фокус

- Безперервна демонстрація цінності,
- функціональність,
- automation,
- AI,
- integration,
- ROI.

### Основні принципи

- Чіткий value proposition
- Показати "До / Після"
- Дані, цифри, графіки
- Use cases
- Докази безпеки, інтеграції
- Pricing expectation

### Генераційний профіль

```text
# PROFILE: SaaS Landing
- Focus on results and automation.
- Use value-based messaging over technical features.
- Include "Before vs After" transformation.
- Add 3–4 use cases for different customer segments.
- Emphasize integrations, API, security (GDPR, ISO).
- Show quantifiable improvements (e.g., -40% costs).
- CTA: “Start Free Trial”, “Request Demo”.
```

## B. E-commerce Landing Page Profile

### Фокус

- Емоції,
- вигода,
- urgency,
- ціна,
- довіра,
- якість.

### Основні принципи

- Вигода + емоція
- Прості блоки
- Великий соціальний доказ
- Reviews
- Bonuses / Offers
- Гарантії

### Генераційний профіль

```text
# PROFILE: E-commerce Landing
- Focus on emotional language + clear benefits.
- Highlight product quality, materials, uniqueness.
- Include social proof: testimonials, ratings, photos.
- Add urgency triggers: limited stock, discount timers.
- Emphasize guarantee, return policy, fast shipping.
- CTA: “Buy Now”, “Add to Cart”, “Get Yours Today”.
```

## C. Consulting / Service Landing Page Profile

### Фокус

- Експертиза,
- досвід,
- довіра,
- процес,
- кейси.

### Основні принципи

- Експертність
- Авторитет
- Кейси з результатами
- Процес роботи
- FAQ з запереченнями
- Discovery Call

### Генераційний профіль

```text
# PROFILE: Consulting/Service Landing
- Emphasize expertise, authority, certifications.
- Show transformations through case studies.
- Clearly outline your process in 3–5 steps.
- Address typical objections in FAQ.
- Add trust elements: experience, awards, publications.
- CTA: “Book Consultation”, “Schedule a Call”.
```

## D. B2B Enterprise Landing Page Profile

### Фокус

- Довгий цикл угоди,
- ризики,
- безпека,
- інтеграції,
- бізнес-вартість.

### Основні принципи

- Конкретні бізнес-ризики
- TCO (Total Cost of Ownership)
- Анкоринг на вартість
- ROI
- Enterprise-grade features
- Повна прозорість

### Генераційний профіль

```text
# PROFILE: B2B Enterprise Landing
- Use authoritative, data-driven tone.
- Highlight enterprise requirements: security, compliance, integrations.
- Provide quantified business value (ROI, TCO reduction).
- Include case studies from similar industries.
- Use risk-reversing statements.
- CTA: “Request Enterprise Demo”, “Talk to Sales”.
```
