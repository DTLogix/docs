# Маркетинг та конверсія

Цей розділ містить матеріали з оптимізації конверсії, створення лендінгів та генерації контенту для DTLogix.

## Структура

- [CRO (Conversion Rate Optimization)](/marketing/cro) — принципи та чеклісти для оптимізації конверсії
- [Профілі лендінгів](/marketing/landings-profiles) — шаблони для різних типів лендінгів (SaaS, E-commerce, Consulting, B2B Enterprise)
- [Conversion Copywriting Prompt Framework](/marketing/ccpf) — фреймворк для AI-генерації контенту

## Навіщо цей розділ?

- Стандартизує підхід до створення лендінгів та маркетингових матеріалів
- Забезпечує високу конверсію завдяки структурованим принципам CRO
- Дозволяє швидко генерувати контент через AI-інструменти
- Служить джерелом інструкцій для команди маркетингу та контент-менеджерів
