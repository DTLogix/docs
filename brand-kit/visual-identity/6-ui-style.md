# Стиль UI

UI має передавати спокій, технологічність і контроль.

## Компоненти

### Кнопки

**Стани кнопок:**
- **Default:** Primary колір (Teal #137C7A), білий текст
- **Hover:** Трохи темніший відтінок primary кольору
- **Active/Pressed:** Ще темніший відтінок
- **Disabled:** Сірий колір (#9CA3AF), знижена прозорість

**Розміри:**
- Large: 48px висота
- Medium (default): 40px висота
- Small: 32px висота

**Принципи:**
- Закруглені кути: 4–6px
- Мінімальна ширина: 120px для primary кнопок
- Іконки можуть додаватися до тексту

### Поля (Inputs)

**Стани полів:**
- **Default:** Білий фон (#FFFFFF), сіра обводка (#D1D5DB)
- **Focus:** Primary колір обводки (#137C7A)
- **Error:** Error колір обводки (#EF4444)
- **Disabled:** Сірий фон (#F3F4F6), сіра обводка

**Принципи:**
- Закруглені кути: 4px
- Висота: 40px (default)
- Padding: 12px горизонтально
- Placeholder текст: сірий (#9CA3AF)

### Карточки (Cards)

**Принципи:**
- Білий фон (#FFFFFF) на світлому фоні сторінки (#F5F7F7)
- Тінь: легка тінь для відокремлення від фону
- Закруглені кути: 8px
- Padding: 16–24px
- Закруглені кути: 8px

**Використання:**
- Групування пов'язаної інформації
- Відображення рейсів, заявок
- Модальні вікна

### Таблиці

**Принципи:**
- Чіткі розділи між рядками (border або alternate row colors)
- Фіксована висота рядків для читабельності
- Закруглені кути: 8px для контейнера таблиці
- Hover стани для рядків (легке підсвічування)

**Заголовки колонок:**
- Напівжирний текст
- Сірий фон (#F9FAFB) або білий з нижньою border

### Модальні вікна

**Принципи:**
- Білий фон на затемненому overlay (backdrop)
- Закруглені кути: 12px
- Мінімальна ширина: 400px, максимальна: 600px (для стандартних модалок)
- Кнопка закриття в правому верхньому куті
- Padding: 24px

## Принципи дизайну

- **Мінімум візуального шуму:** Використовувати тільки необхідні елементи, уникати зайвих прикрас
- **Чисті контури:** Чіткі межі між елементами, достатній контраст
- **Обмежений набір кольорів:** Використовувати тільки кольори з палітри (див. [Кольори](./2-color.md))
- **Чітка ієрархія:** Використовувати типографіку та spacing для створення ієрархії (див. [Типографіка](./3-typography.md))

Детальніше про дизайн-систему: [/brand-kit/design-system.md](/brand-kit/design-system.md).
