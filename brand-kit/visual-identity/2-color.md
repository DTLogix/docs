# Кольори

Кольори підтримують емоційний профіль бренду: чистота, ясність, структурність.

## Основна палітра

| Роль       | Колір   | HEX      | Використання                           |
|-----------|---------|----------|----------------------------------------|
| Primary   | Teal    | `#137C7A`| основні кнопки, акценти, ключові елементи |
| Secondary | Amber   | `#EFBF45`| індикатори статусів, підсвітка важливих блоків |
| Background| Light   | `#F5F7F7`| фон сторінок, великі площини           |
| Surface   | White   | `#FFFFFF`| картки, модальні вікна, таблиці        |
| Text      | Dark    | `#111827`| основний текст                         |

## Функціональна палітра

| Роль     | HEX      | Використання                    |
|----------|----------|---------------------------------|
| Success  | `#22C55E`| успішні операції, “OK” стани    |
| Warning  | `#FACC15`| затримки, потенційні ризики     |
| Error    | `#EF4444`| помилки, критичні стани        |
| Info     | `#3B82F6`| додаткова інформація, підказки |

## Принципи використання

- мінімалізм
- чіткі акценти
- фонова спокійна палітра
- primary використовується дозовано, для дій та ключових елементів
- secondary — для підсвітки, а не як основний колір
- functional кольори застосовуються тільки за своїм призначенням (статуси, повідомлення)
