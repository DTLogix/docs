# Технологічний стек та інструменти розробки

Цей документ містить посилання на технології, фреймворки, бібліотеки та інструменти, які використовуються для розробки DTLogix.

## Документація

### VitePress

- **[VitePress Guide](https://vitepress.dev/guide/)** - Документація VitePress, використовується для створення цієї документації

## Backend / CMS

### Directus

- **[Directus](https://github.com/directus/directus)** - Основний репозиторій Directus (headless CMS)
- **[Directus Website](https://github.com/directus/website)** - Офіційний сайт Directus
- **[Awesome Directus](https://github.com/directus-labs/awesome-directus)** - Колекція ресурсів та інструментів для Directus
- **[Directus Starters](https://github.com/directus-labs/starters)** - Готові стартові проекти для Directus
- **[Directus Templates](https://github.com/directus-labs/directus-templates)** - Шаблони для Directus
- **[Agency OS](https://github.com/directus-labs/agency-os)** - Template для агентств на базі Directus
- **[Directus Guest Authoring](https://github.com/directus-labs/directus-guest-authoring)** - Гостьова авторизація для Directus
- **[Directus Railway CMS Template](https://github.com/directus-labs/Directus-Railway-CMS-Template)** - Шаблон для розгортання Directus на Railway
