# Інтеграції DTLogix

## Вступ

Цей документ описує інтеграції DTLogix з зовнішніми сервісами. Всі інтеграції базуються на планах з roadmap та strategy (див. [/strategy/roadmap.md](/strategy/roadmap.md)).

## Інтеграції для MVP / V1

### SMS інтеграції

**Призначення:** Відправка SMS сповіщень клієнтам про зміни статусу рейсів та ETA.

**Статус:** Планується для MVP / V1.

**Використання:**
- Сповіщення про зміну статусу рейсу
- Сповіщення про зміну ETA
- Сповіщення про доставку

**Зв'язок з модулями:** Інтегрована з модулем "Сповіщення" (див. [/product/modules.md](/product/modules.md#модуль-сповіщення)).

---

### Email інтеграції

**Призначення:** Відправка email сповіщень клієнтам.

**Статус:** Планується для MVP / V1.

**Використання:**
- Сповіщення про зміну статусу рейсу
- Сповіщення про зміну ETA
- Сповіщення про доставку
- Email з клієнтським лінком

**Зв'язок з модулями:** Інтегрована з модулем "Сповіщення".

---

## Інтеграції для V2

### GPS інтеграції

**Призначення:** Автоматичне відстеження місцезнаходження вантажу через GPS.

**Статус:** Планується для V2.

**Використання:**
- Автоматичне оновлення статусу рейсу
- Покращення точності ETA на основі реального місцезнаходження
- Інтеграція з мобільним застосунком для водіїв

**Зв'язок з модулями:** Інтегрована з модулями "Рейси" та "ETA".

---

### API / Webhooks

**Призначення:** Програмний інтерфейс для інтеграції DTLogix з іншими системами.

**Статус:** Планується для V2.

**Використання:**
- Інтеграція з іншими логістичними системами
- Автоматичне створення рейсів з зовнішніх систем
- Webhooks для отримання подій у реальному часі

**Зв'язок з модулями:** Інтегрована з Core API та усіма модулями.

---

### Бухгалтерія

**Призначення:** Інтеграція з бухгалтерськими системами для автоматичного формування документів та звітів.

**Статус:** Планується для V2.

**Використання:**
- Автоматичне формування документів
- Синхронізація даних про рейси з бухгалтерією
- Генерація звітів

---

### Маркетплейси вантажів

**Призначення:** Інтеграція з маркетплейсами вантажів для автоматичного створення заявок та рейсів.

**Статус:** Планується для V2.

**Використання:**
- Імпорт заявок з маркетплейсів
- Автоматичне створення рейсів
- Синхронізація статусів

---

## Корисні посилання для розробки

Список зовнішніх API, сервісів та нормативно-правових документів, які можуть бути потрібні для розробки CRM системи DTLogix.

### Державні API та сервіси

#### Митні сервіси України

- **[Реєстр ЄКМТ](https://middleware.dsbt.gov.ua/cemt/reg)** - Реєстр ЄКМТ-дозволів для міжнародних перевезень
- **[Ліцензії на перевезення](https://middleware.dsbt.gov.ua/lc/licenses)** - API для отримання інформації про ліцензії на перевезення

#### Електронна черга

- **[Електронна черга (eCherha)](https://echerha.gov.ua/workload/1/1)** - Сервіс для запису та відстеження черги на митниці

#### Кордон

- **[Польська митниця (Granica)](https://granica.gov.pl/index_wait.php?p=u&v=pl&k=w)** - Інформація про чергу на польсько-українському кордоні

### API логістичних платформ

#### Lardi-Trans API

- **[Lardi-Trans API](https://lardi-trans.ua/larditransapi/)** - API для інтеграції з платформою Lardi-Trans. Дозволяє додавати та редагувати заявки щодо вантажу та транспорту, отримувати дані з довідників, повторювати заявки та будувати маршрути.

### Фінансові API

#### Національний банк України

- **[API НБУ - Курси валют](https://bank.gov.ua/ua/open-data/api-dev)** - API для отримання офіційних курсів гривні до іноземних валют та СПЗ, облікових цін банківських металів. Використовується для автоматичного оновлення валютних курсів у системі.

### Реєстри та перевірка контрагентів

#### OpenDataBot API

- **[OpenDataBot API](https://opendatabot.ua/open/api)** - API для доступу до державних реєстрів України. Надає інформацію про учасників ринку транспортно-експедиторської діяльності, включаючи:
  - Єдиний державний реєстр (ЄДР)
  - Єдиний державний реєстр боржників (ЄДРБ)
  - Єдиний державний реєстр судових рішень (ЄРСР)
  - Перевірка контрагентів
  - Інші державні реєстри

### Нормативно-правова база

#### Закони України

- **[Про транспортно-експедиторську діяльність](https://zakon.rada.gov.ua/laws/show/1955-15#Text)** - Закон України від 01.07.2004 № 1955-IV, який регулює транспортно-експедиторську діяльність в Україні
