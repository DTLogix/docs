# Ключові метрики (Key Metrics)

## Product Metrics
- MAU / DAU (активні користувачі).
- Кількість оброблених рейсів.
- % рейсів, повністю закритих документами та платежами.

## Business Metrics
- MRR / ARR.
- Churn rate.
- LTV / CAC.

## Value Metrics (цінність для клієнта)
- Скорочення часу обробки заявки.
- Зменшення логістичних помилок.
- NPS.
