# Цільові сегменти клієнтів

## Вступ

Цільові сегменти визначають, кому продукт створює найбільшу цінність та на кого варто спрямувати маркетинг і функціональні рішення.

## Детальний опис

### Сегмент A

**Соло-експедитори / малі команди (1–5 осіб)**:

- 20–60 рейсів/місяць  
- високий рівень ручної роботи  
- потреба у швидкому впровадженні та простому інтерфейсі  

### Сегмент B

**Малі логістичні компанії (до 100 рейсів/місяць)**:

- хаос у комунікаціях  
- важливість історії рейсів та контролю SLA  
- часті помилки через багатозадачність  

### Сегмент C

**Клієнти експедиторів**:

- очікують прозорості та прогнозованого ETA  
- хочуть отримувати сповіщення без дзвінків  

## Гіпотези

- Найшвидше конвертується сегмент A, бо проблема найбільш відчутна.  
- Сегмент B забезпечить більший LTV через регулярний об’єм рейсів.  
- Клієнтам експедиторів (сегмент С) критично важливі сповіщення та ETA → вони впливають на adoption.

## Наступні дії / наслідки
- Маркетинг фокусується на сегменті A для швидкої валідації.  
- Для сегменту B — демонстрації та кастомні onboarding-скрипти.  
- Для сегменту C — створення окремого клієнтського лінку з CTA.
